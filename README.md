# StackFit

StackFit is a smart web application that recommends the perfect tech stack for your project based on budget, team size, and requirements. It provides AI-powered reasoning, architecture diagrams, cost estimates, and migration paths.

## Features
- AI-Powered Reasoning
- Architecture Diagrams
- Real cost & scale estimates
- Stack Comparisons
- Migration Guides

## Setup
```bash
npm install
npm run dev
```
